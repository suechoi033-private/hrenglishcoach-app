## 운영 플로우 (현재 — 수동)

1. 사용자가 앱의 "이메일로 구독 신청" 클릭 → sue.choi033@gmail.com 으로 메일 수신
2. 결제 안내(계좌이체 또는 추후 결제링크) 회신
3. 입금 확인 후 **라이선스 코드** 1개 발송 (코드 목록은 운영자가 별도 보관 — 저장소에 커밋 금지)
4. HR 인증 신청 메일(명함 첨부) → 확인 후 할인가 안내 + 코드 발송

## 라이선스 코드

- 형식: `HREC-XXXX-XXXX`
- 앱에는 코드의 SHA-256 해시만 포함 (`pro.js`의 `PRO_HASHES`)
- 새 코드 발급: 코드를 만들고 `echo -n "HREC-...." | shasum -a 256` 해시를 `PRO_HASHES`에 추가 후 배포
- 사용한 코드는 목록에서 지우지 않아도 되지만, 유출 시 해당 해시 삭제로 무효화 가능

    사용법

    1. 난이도 선택 — Beginner / Intermediate / Advanced 중 선택하세요.
    2. Briefing & Case Study 읽기 — 🔊 버튼으로 듣고, 모르는 단어는 클릭하면 사전이 열립니다.
    3. 질문에 답변 — 텍스트로 쓰거나 🎤 음성으로 녹음하세요.
    4. 모범 답변과 비교 — 제출 후 자신의 답변과 비교하고 답변 팁을 확인하세요.
    5. 학습 기록 — 모든 답변은 브라우저에 자동 저장됩니다 (서버 전송 없음).

    음성 기능 사용을 위해서는:

    • 🎤 음성 입력: Chrome 또는 Edge 브라우저 필요 (Safari/Firefox는 일부 제한)
    • 🔊 음성 출력: 모든 최신 브라우저에서 작동
    • HTTPS 환경 필요 (GitHub Pages 또는 localhost)
